РЕФЕРАТ ПРОГРАММЫ ДЛЯ ЭВМ

Название: «MyPillBox — веб-сервис персонального контроля приёма лекарственных средств»

Программа представляет собой веб-сервис, предназначенный для автоматизации контроля
за приёмом лекарственных средств, верификации их подлинности и поиска аптек
на интерактивной карте. Сервис реализован в виде одностраничного веб-приложения
(SPA) с серверной частью, обеспечивающей обработку данных.

Программа обеспечивает следующие функции: сканирование DataMatrix-кодов маркировки
лекарственных препаратов с помощью камеры устройства и получение верифицированных
сведений о препарате (подлинность, срок годности, производитель) через интеграцию
с государственной системой «Честный ЗНАК»; геолокационный поиск аптек в радиусе
2 км от пользователя с отображением на интерактивной карте и сортировкой
по расстоянию; ведение персонального журнала лекарственных средств с возможностью
аутентификации по электронной почте.

Программа реализована с использованием библиотеки React (JavaScript) на стороне
клиента и фреймворка FastAPI (Python) на стороне сервера. Хранение пользовательских
данных осуществляется посредством облачной СУБД Supabase (PostgreSQL).

Область применения: персональное здравоохранение, контроль лекарственной
безопасности, фармацевтическая отрасль.

Объём программы: ~990 строк исходного кода.
